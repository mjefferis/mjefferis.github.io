Hi there, name's Matt. I'm a Philly fullstack web developer. Take a look at some of my projects.

Tune Tracker

This app was made to help musicians connect. It uses both mapbox's and youtube's APIs to let users upload their geographical coordinates and a clip of their music. Using Firebase, user information is saved onto the app.

Phish Quiz

Built using JavaScript and jQuery, the user has 60 seconds to answer five questions about the rock band Phish. Once all questions are answered or the time is up, the number answered correctly is tallied.

Twin Peaks Gifs Generator

This webpage uses GIPHY's API to feature GIFS from Twin Peaks. Click on a character button and 10 GIFS corresponding to the character will be displayed.

Train Scheduler

Using Firebase and moment.js, users upload train information and see the next departure time of the train in real time.

Bamazon

An Amazon-like storefront built with MySql and Node.js. The app takes in orders from users and deplete stock from the store's inventory. Customers can also add items to inventory and view items low in stock.

Liri

A Language Interpretation and Recognition Interface. LIRI is a command line node app. Calling upon the Twitter, Spotify, and OMDB API, it will give you data regarding my latest tweets and also song and fim information.

Crystal Game

This app generates a target number and the user is to match that number by clicking crystals. Each crystal is given a random numerical value between 1 and 12. The user wins if they match the number.

Psychic Game

The app generates a random letter and the user has nine chances to guess the correct letter. If the user either guesses correct or runs out of guess, a win or loss is tallied and the game automatically resets.
