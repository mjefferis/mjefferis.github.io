Hi there, name's Matt. I'm a Philly fullstack web developer. Take a look at some of my projects.

Tune Tracker

This app was made to help musicians connect. It uses both mapbox's and youtube's APIs to let users upload their location and a clip of their music. Using Firebase, user information is saved and viewable on any machine.

Phish Quiz

Built using Javascript and jQuery, the user has 60 seconds to answer five questions about the rock band, Phish. Once all questions are answered or the time is up, the number correct is tallied and displayed to the user.

Twin Peaks Gifs Generator

This webpage uses GIPHY's API to feature GIFS from Twin Peaks. Click on a character button and 10 GIFS corresponding to the character will be displayed.

Train Scheduler

Using Firebase and moment.js, users can upload train information and see the next departume time of the train in real time.

Bamazon

An Amazon-like storefront with built with MySql and Node.js. The app takes in orders from customers and deplete stock from the store's inventory.

Liri

A Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data. Calls upon the Twitter, Spotify, and OMDB API.

Crystal Game

The app generates a target number and the user is to match that number by clicking crystals. Each crystal is given a random numerical value between 1 and 12. If the user goes over the target number, they lose, whereas they win if they match the number.

Psychic Game

The app generates a random letter and the user has nine chances to guess the correct letter. If the user guesses correct, a win is tallied and the game resets. If the user runs out of guesses, a loss is tallied and the app resets.
